STEP 1: m4brew converts your audiobooks into a single chapterised .m4b.

STEP 2: Rename your new .m4b files based on folder structure (see below).

STEP 3: Nothing is lost. Delete the original files ONLY when you’re ready.

Converting

Test: shows which books will be converted.

Run: begins the actual conversion process.

Renaming

Renaming is based on folder structure...

author_name/book_title becomes the name: book - author.m4b

Deleting

If you are finished and you want to free up disc space...

This task deletes all the original files.

What file types are converted?
  • Single mp3 → single m4b
  • Multiple mp3 → single m4b
  • Single m4a → single m4b
  • Multiple m4a → single m4b
  • Multiple m4b → single m4b
How are chapters created?

Chapters are defined by source files.

Every individual file becomes its own chapter in the final converted .m4b

Chapter names are also defined by the names of the source files.

Folder structure: IMPORTANT!
Audiobooks/
└── Author/
    └── Book Title/
        └── audio files
What about original files?

NOTHING IS DELETED.

After conversion all the original files are stored safely in a new folder:

book folder/_backup_files
