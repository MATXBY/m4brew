Intro
m4brew is a friendly, opinionated tool for audiobook housekeeping. It turns messy folders into clean, chapterised .m4b files — consistent names, predictable results.
Contents
Keep your library in a simple structure so m4brew can work reliably:
Audiobooks / Author / Book / files…
(One book per folder.)
The Brewer
It’s a three-step process:
  • Step 1 — convert folders into a single .m4b
  • Step 2 — rename existing .m4b files to match the folder structure
  • Step 3 — remove _backup_files once you’re happy
Jobs run in the background — you can switch pages safely. Start with Test, then hit Start.
Epilogue
m4brew won’t guess wildly — it aims to be predictable and reversible. If you’re ever unsure, run Test first.
