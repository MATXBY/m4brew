Step 1: m4brew converts your audiobooks into a single chapterised .m4b.

Step 2: Rename your new .m4b files based on folder structure (see below).

Step 3: Nothing is lost. Delete the original files ONLY when you’re ready.

Task 1 - Conversion

Test shows which books are going to be converted.

Run begins the actual conversion session.

Task 2 - Renaming

Renaming is based on folder structure:

author / book / files

This renames the m4b:

book-author.m4b

Task 3 - Delete

When you’re happy with all your new .m4b files you can delete the original mp3/m4a files.

What file types are converted?
  • Single mp3 → single m4b
  • Multiple mp3 → single m4b
  • Single m4a → single m4b
  • Multiple m4a → single m4b
  • Multiple m4b → single m4b
How are chapters created?

Chapters are based on individual source files. Each file becomes a chapter in the final .m4b. Chapter names are based on source files.

What is the folder structure?

ESSENTIAL:

root_folder/

author_name/

book_name/

files

What about original files?

NOTHING IS DELETED. After conversion all original files are moved to a separate _backup_files folder next to the newly created .m4b.
