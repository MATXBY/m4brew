Introduction

m4brew converts audiobooks into a single .m4b, correctly named and chapterised based on your folder structure.

This makes imports and metadata management easier when importing audiobooks into your preferred library/management software.

Tasks
Task 1 — Conversion
  • Test: a practice run — shows what would be actioned
  • Run: actual conversion of the files
  • Toggle live output on/off for detailed view
Task 2 — Renaming

Renames books based on the folder structure:

root/author name/book name → author name - book name.m4b

Task 3 — Delete

When you’re happy with all of the new m4b files you can delete the backed up original files.

Conversion Overview
  • Single mp3 books into a single m4b
  • Multiple mp3 books into a single m4b
  • Single m4a books into a single m4b
  • Multiple m4a books into a single m4b
  • Multiple m4b files into a single m4b
Folder structure

ESSENTIAL: root/author name/book name/files

Settings
  • Select your audiobook root folder
  • Select audio channels — mono / stereo / match existing
  • Select bitrate — specific bitrate / match existing
Chapters

Chapters are based on the individual source files that make up the m4b. Each file becomes a chapter and is named according to the source file.

Original files

These are NOT deleted. After conversion they are moved to a _backup_files folder within the corresponding book folder.
