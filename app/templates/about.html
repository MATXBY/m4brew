Welcome

M4Brew is a simple tool for converting and organising audiobooks into clean, chapterised M4B files with AAC audio, making them easier to manage and more reliable for media players like Plex, Jellyfin, and Audiobookshelf.

It’s designed to handle the fiddly parts of audiobook housekeeping while staying safe, predictable, and transparent.

Folder structure (important)

Audiobooks/
└── Author Name/
    └── Book Title/
        └── audio files

M4Brew expects this structure. It allows output files to be named consistently, generates reliable chapter titles, and helps media servers correctly identify books and authors.

Supported input formats

  • MP3 (single or multiple files)
  • M4A (single or multiple files)
  • Existing M4B (merged into one)

Output is always a single, chapterised .m4b (AAC) file.

Safety first

  • Original audio files are never deleted automatically
  • Converted files are created alongside your existing structure
  • Cleanup is a separate, explicit step
  • Jobs can be safely cancelled — no half-written files are left behind
